# Product Sense

## Mission

Help&Grow connects ambitious startup founders in Singapore and Southeast Asia with seasoned domain experts for paid consulting sessions, powered by AI matching.

## User Personas

### Founder (Buyer)
- Early-stage startup founder in SEA
- Needs specific domain expertise (legal, fundraising, product, marketing, tech)
- Values speed — wants to find and book the right expert quickly
- Price-sensitive but willing to pay for quality advice
- Uses Telegram (primary), WeChat (Chinese-speaking), or web browser

### Expert (Seller)
- Experienced professional with domain expertise
- Wants to monetize knowledge through consulting
- Needs minimal friction to set up profile and receive bookings
- Expects reliable payouts and professional booking management

## Product Principles

1. **AI-first discovery**: Founders describe their problem in natural language; AI matches them to the best expert
2. **Multi-platform parity**: Same core experience on web, Telegram, and WeChat
3. **Trust through structure**: Verified profiles, transparent pricing, two-way reviews
4. **Minimal friction**: Expert onboarding in < 5 minutes (AI generates profile from social links)
5. **Fair marketplace**: 50% deposit model protects both sides; platform fee is transparent

## Key User Journeys

### Founder: Find → Book → Review
1. Describe need via AI match or browse expert list
2. View expert profile (bio, domains, pricing, reviews, availability)
3. Select time slot(s) and pay 50% deposit
4. Attend session (online via meeting link or offline)
5. Leave review after session

### Expert: Onboard → Get Booked → Get Paid
1. Sign up, connect social profiles
2. AI generates professional bio and profile image
3. Set availability, pricing, and session types
4. Complete Stripe KYC for payouts
5. Receive booking notifications, conduct sessions
6. Remainder auto-charged 24h after session; payout via Stripe
